Preppings Search Bar widget for food bloggers.

== Description ==

Preppings Search Bar Plugin adds a widget to put into blog layout. Preppings Search Bar is a searching tool
over Preppings.com food database.

== Installation ==

Plugin can be installed by two ways:

* From ZIP file.
* Directly from Wordpress Plugins Administration Panel.

= Installation from ZIP file =

Download latest plugin ZIP-file from http://preppings.com/preppings-search-bar.zip.
Unpack it into `/wp-content/plugins/` directory

= Installation from Wordpress Plugins Administration Panel =

Go to Add New of your Wordpress Plugins Admin Panel `/wp-admin/plugin-install.php` and search for `Preppings Search Bar`.
Click on 'Install Plugin' and activate it after installation.

= Add a widget =

After installation add Preppings Search Bar widget to your Wordpress theme layout.
To use widget you need to specify Preppings API Key. The key can be obtained from http://preppings.com/api/
or directly from widget customization dialog.
